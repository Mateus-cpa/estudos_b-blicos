# 📖 Estudos Bíblicos
>"Lâmpada para os meus pés é tua palavra, e luz para o meu caminho." — Salmos 119:105

Este repositório é dedicado ao armazenamento, organização e compartilhamento de materiais de estudos bíblicos produzidos por mim. O objetivo é criar um acervo pessoal de aprendizado, reflexões e exegese, facilitando o acesso e a revisão constante das Escrituras.
